# go-timetrack

A simple terminal time tracker built with Cobra.

## Install `tt` globally

From the project root:

```bash
go install .
```

Make sure your Go bin is in your `PATH` (usually `$HOME/go/bin`):

```bash
echo 'export PATH="$HOME/go/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

Verify:

```bash
tt --help
```
